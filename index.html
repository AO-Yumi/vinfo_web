vinfo - bitrate analyzer for Audio file

    This tool is for displaying the information in the vorbis, opus, (ogg)flac and/or other ogg container file, and mp3, aac(adts format).

    このソフトウェアはogg vorbisやその他のoggコンテナを用いたファイル、及びmp3やaac(adts形式)の情報を表示することができるツールです。

    Changes
    # Added display of information of MP3 and AAC(ADTS format) files.
    # MP3 and AAC(ADTS format): Enabled "-ab" option.
    # Opus and OggFLAC: Added display of "Page/Packet Info".
    # Opus and OggFLAC: Improved the processing speed.
    # Vorbis: Fixed handling of very small file.
    # Fixed method of caluculation of average bitrate in "-ab" option.
    更新履歴
    □mp3（MPEG-1/2/2.5 Audio Layer3）、aac（ADTS形式）の情報表示に対応
    □-abオプションにてmp3、aac(ADTS形式)ファイルのビットレート分析に対応
    □Opus、OggFLACの"Page/Packet Info"をVorbisとほぼ同等の内容とした
    □OpusとOggFLACの処理速度を改善
    □-abオプションで出力されるファイルの平均ビットレート値をaverage stream bitrate値と同じにした
    □vorbisにおいて、非常に小さいファイルのサンプル値が異常な値になることがあったのを修正（v0.8からのバグ）
